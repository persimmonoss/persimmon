# Persimmon Browser

The Persimmon Browser is a **fast, privacy-focused, Chromium-based browser** built with Electron and React.  
It is designed to provide a **cool looking, functional, and secure browsing experience** for all users.

## Features

- Privacy-focused: tracker & ad blocker, private mode, no telemetry
- Lightweight and fast: efficient tab management, GPU acceleration
- Cool UI: minimalist design, themes, smooth animations
- Built with Electron, React, and styled-components

## Installation

```bash
git clone https://github.com/persimmonoss/persimmon.git
cd persimmon-browser
npm install
npm start
```
